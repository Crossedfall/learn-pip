version number: 0.0.1

Overview
--------

A python package that can be installed with pip.

Installation / Usage
--------------------

To install use pip:

    $ pip install mypippkg


Or clone the repo:

    $ git clone https://github.com/crossedfall/mypippkg.git
    $ python setup.py install
